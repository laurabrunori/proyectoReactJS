Shop Brunori

A continuación podrán leer una breve reseña de la aplicación web desarrollada en React y Vite como proyecto final del curso de ReactJS. La misma permite visualizar un catálogo de productos, registrarse e iniciar sesión mediante Firebase Authentication, administrar productos utilizando Firestore y realizar compras mediante un carrito de compras implementado con Context API.

---

TECNOLOGIAS UTILIZADAS:
- React.
- Vite.
- React Router DOM.
- Firebase Authentication.
- Firebase Firestore.
- Bootstrap 5.
- Context API.
- JavaScript.

---

FUNCIONES:

Autenticación:
- Registro de usuarios.
- Inicio de sesión.
- Cierre de sesión.
- Rutas protegidas.
- Perfil de usuario.

Carrito de compras
- Agregar productos nuevos.
- Eliminar productos.
- Modificar la cantidad de productos.
- Vaciar carrito.
- Calcular el precio total.
- Agregar cupones de descuento.

Gestión de productos (CRUD) Desde el panel de administración es posible:
- Crear productos.
- Editar productos.
- Eliminar productos.
- Visualizar el listado completo.
- Confirmar eliminación mediante un modal.
(Los datos son almacenados en Firebase Firestore).
